# Manual de Uso: AutoCloudPC OS (v1.0-Alpha)

Bienvenido a la documentación oficial de **AutoCloudPC OS**, el sistema que transforma tu Android en una experiencia de escritorio Windows 10 ultra-rápida.

## 🚀 Instalación
1. Clona este repositorio o descarga el código fuente.
2. Abre el proyecto en **Android Studio**.
3. Compila la APK y asegúrate de otorgar los permisos de almacenamiento cuando se soliciten.
4. Al abrir la app por primera vez, el sistema extraerá automáticamente el motor de emulación.

## ⚙️ Optimizaciones Incluidas
AutoCloudPC OS no es un emulador común; incluye tecnologías de vanguardia:
- **DynaCache**: La traducción de programas se guarda en disco para que la carga sea instantánea tras el primer uso.
- **ntsync**: Sincronización a nivel de kernel para un ahorro masivo de CPU y batería.
- **Auto-GPU**: Detección automática de drivers para Mali y Adreno.
- **Low-Latency Audio**: Configuración de PulseAudio optimizada para Android.

## 🖱️ Controles
- **Un toque**: Clic izquierdo.
- **Dos dedos**: Clic derecho.
- **Doble toque y arrastrar**: Mover ventanas o seleccionar archivos.
- **Teclado**: Se integra con el teclado virtual de Android o teclados físicos vía USB/Bluetooth.

## 📂 Estructura de Archivos
- `C:\`: Mapeado a la carpeta interna de la APK para máxima velocidad.
- `D:\`: Mapeado a tu `/sdcard` de Android para que accedas a tus archivos personales.
